# Zaish Denim

Small-batch denim and design workshops for universities and teams.

## Features
- **Modern Design**: Clean, responsive layout for desktop and mobile.
- **Shop**: Interactive product showcase.
- **Workshops**: Direct inquiry system for institutions.
- **Newsletter**: Simple sign-up for updates.

## Setup & Running

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the site**:
   ```bash
   npm start
   ```
   This will serve the site at `http://localhost:5000` (or the next available port).
